# Todo App

Современное приложение для управления задачами с полноценной авторизацией.

## О проекте

Backend-приложение на **NestJS**, которое позволяет пользователям регистрироваться, авторизовываться и управлять своими задачами.

### Основные возможности

- Регистрация и авторизация (JWT + Refresh Token)
- Создание, просмотр, редактирование и удаление задач
- Фильтрация задач по статусу, приоритету и дедлайну
- Пагинация и сортировка
- Soft Delete задач
- Защита всех роутов (каждый пользователь видит только свои задачи)

## Технологический стек

- **Backend**: NestJS + TypeScript
- **ORM**: Prisma
- **База данных**: SQLite (готов к переходу на PostgreSQL)
- **Авторизация**: JWT + Refresh Token
- **Валидация**: class-validator + class-transformer
- **Документация**: Swagger (OpenAPI)
- **Контейнеризация**: Docker + Docker Compose

## Запуск проекта

### 1. Локально

```bash
# Клонирование репозитория
git clone https://github.com/MagaSabir/todo-app.git
cd todo-app

# Установка зависимостей
npm install

# Генерация Prisma Client
npx prisma generate

# Применение миграций
npx prisma migrate dev

# Запуск в режиме разработки
npm run start:dev
